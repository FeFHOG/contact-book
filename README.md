# 通讯录管理系统

本项目是 C++ 课程设计程序，使用面向对象方法实现手机和手机卡两个位置的通讯录管理。

## 功能

- 新建联系人
- 删除联系人
- 修改联系人
- 按姓名查询联系人
- 浏览所有联系人
- 手机和手机卡之间移动联系人
- 手机和手机卡之间复制联系人
- 使用文本文件保存联系人数据

## 面向对象特点

- `ContactPhone` 继承 `ContactCard`
- `ContactBookInterface` 定义通讯录多态接口
- `ContactBookCard` 和 `ContactBookPhone` 实现具体通讯录
- 重载 `>>` 和 `<<` 完成联系人输入输出
- `User` 类组合两个通讯录并完成业务操作

## 编译运行

```bash
make run
```
